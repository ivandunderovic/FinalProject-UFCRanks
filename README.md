# UFC-Ranks
Web application for ranking UFC fighters
## Project Description
UFC Ranks is a web application that allows users to view top 10 UFC fighters from each weight category.
This project was created as a final project for the course "Osnove razvoja Web i mobilnih aplikacija".

## Project features
- fighter rankings
- responsive design
- dynamic data display
- form validation

## Technologies used
- HTML
- CSS
- JavaScript
